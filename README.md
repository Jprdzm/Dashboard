# Second Brain — Dashboard Personal

Dashboard estilo Notion con temporizador Pomodoro, registro de gastos, calendario y más.

## Correr localmente

```bash
npm install
npm run dev
```

## Desplegar en Vercel

```bash
npm run build
npx vercel --prod
```

O conecta el repositorio directamente desde [vercel.com/import](https://vercel.com/import).
